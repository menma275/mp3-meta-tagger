# mp3metatagger

複数MP3の一括アルバム化および同期歌詞エディター。

- **デプロイ先**: [https://mp3-meta-tagger.vercel.app/](https://mp3-meta-tagger.vercel.app/)

## 主な機能

- 複数MP3ファイルの一括メタデータ（アルバム名、アーティスト、カバーアート等）の編集
- 同期歌詞の編集および埋め込み
- 編集済みMP3ファイルの一括ZIPダウンロード

## 開発・実行方法

### 依存関係のインストール

```bash
pnpm install
```

### 開発サーバーの起動

```bash
pnpm dev
```

### ビルド

```bash
pnpm build
```
